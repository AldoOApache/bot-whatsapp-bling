# Bot WhatsApp + Bling ERP

Bot inteligente que responde clientes no WhatsApp, integrado com seu Bling ERP.

## Funcionalidades

✅ Responde perguntas sobre produtos
✅ Consulta preços em tempo real do Bling
✅ Verifica disponibilidade de estoque
✅ Avisa sobre entregas (Uber, frete normal)
✅ Passa para humano quando cliente alega defeito/garantia
✅ Tone amigável e confortável
